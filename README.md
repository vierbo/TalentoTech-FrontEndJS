## Grave | Tienda para bajistas

Este es un proyecto de prueba para un curso de desarrollo frontend. Es una simulación de una tienda de comercio electrónico especializada en bajos y accesorios para bajistas.

El objetivo es cumplir con los requisitos de la preentrega del curso, utilizando HTML5 y CSS3.

### Tecnologías utilizadas
*   **HTML5**: Para la estructura semántica del contenido.
*   **CSS3**: Para los estilos, la paleta de colores y el diseño responsivo. Se utiliza Flexbox para organizar los elementos de forma adaptable a diferentes tamaños de pantalla.

### Características implementadas
*   Página de inicio con sección de productos.
*   Página de contacto con formulario funcional a través de Formspree.
*   Diseño responsivo que se adapta a dispositivos móviles, incluyendo un menú de hamburguesa.
*   Uso de Google Fonts para una tipografía personalizada.

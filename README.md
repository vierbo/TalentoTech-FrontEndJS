## Grave | Tienda para bajistas

Este es mi proyecto para el curso de desarrollo frontend JS de TalentoTech. 
Es una simulación de una tienda de comercio electrónico.
El objetivo es cumplir con los requisitos de la etapa de preentrega del curso, utilizando HTML5 y CSS3.

### Tecnologías utilizadas

* **HTML5**: Para la estructura semántica del contenido.
* **CSS3**: Para los estilos, la paleta de colores y el diseño responsivo. Se utiliza Flexbox para organizar los elementos de forma adaptable a diferentes tamaños de pantalla.

### Características implementadas

* Página de inicio con sección de productos.
* Página de contacto con formulario funcional a través de Formspree.
* Diseño responsivo que se adapta a dispositivos móviles.
* Uso de Google Fonts para una tipografía personalizada.
